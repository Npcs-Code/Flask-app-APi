FLask API

To Scraping All Elements of a Specific Page :

1 > by number Of page.

Example Code

fetch('http://127.0.0.1:5000/get-page-home-items/1')
.then(res => res.json())
.then(json => console.log(json))

2 > by Category Of elements.

Example Code

fetch('http://127.0.0.1:5000//get-page-category/{Category-Name}/page/{number-page}')
.then(res => res.json())
.then(json => console.log(json))

To scraping deteile element :

1 > by name Of element.

Example Code

fetch('http://127.0.0.1:5000/get-page-single-item/{Name}')
.then(res => res.json())
.then(json => console.log(json))

To scraping Images of a specific Chaptre :

1 > by name or number Of Chaptre.

Example Code

fetch('http://127.0.0.1:5000/get-page-single-chapter/{name}/{chapter}')
.then(res => res.json())
.then(json => console.log(json))
